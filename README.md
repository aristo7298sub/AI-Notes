# AI Notes

> 个人 AI 知识空间——博客、深度解析、学习笔记、行业观察、工具指南、项目实践。

## 目录结构

| 目录                      | 说明                                         | 语言     |
| ------------------------- | -------------------------------------------- | -------- |
| `deep-dives/`             | 原创深度技术解析                             | 中文     |
| `commentaries/`           | 外部文章双语解读（英文原文 + 逐段翻译/见解） | 中英双语 |
| `insights/`               | 行业趋势随笔、观点                           | 中文     |
| `learning-notes/papers/`  | 论文笔记                                     | 中文     |
| `learning-notes/courses/` | 课程笔记                                     | 中文     |
| `learning-notes/books/`   | 书籍笔记                                     | 中文     |
| `tool-guides/`            | 工具教程、配置指南                           | 中文     |
| `projects/`               | 项目实践记录                                 | 中文     |

## 快速开始

```powershell
# 创建新文章（交互式选择模板）
.\_scripts\new-article.ps1

# 重新生成索引目录
.\_scripts\build-catalog.ps1
```

## 写作约定

- 每篇文章必须包含 YAML frontmatter（`title`, `date`, `tags`, `category`）
- 外部解读文章：保留英文原文，逐段翻译 + 个人见解
- 原创文章：中文撰写
- 文件名：小写英文 + 连字符，如 `agent-skills-deep-dive.md`
- 模板见 `_templates/` 目录
